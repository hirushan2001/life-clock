# Life Clock Dashboard

## Project info

A dashboard application to track life progress and goals.

## How to run

1.  **Install dependencies:**
    ```sh
    npm install
    ```

2.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
